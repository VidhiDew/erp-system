# erp-system
 
Project Overview :
This is a ERP (Enterprise Resource Planning) System developed using Spring Boot.
The project provides REST APIs to manage employee data and perform CRUD operations efficiently.

Tech Stack :
Java 8,
Spring Boot,
Spring Data JPA,
Hibernate,
MySQL,
REST API,
Maven,

Project Features :
✔ Employee Management Module
✔ Create, Read, Update, Delete (CRUD) APIs
✔ Exception Handling (Global)
✔ Layered Architecture (Controller, Service, Repository)
✔ Clean Code Structure

Project Structure :
erp-system 
│── controller
│── service
│── repository
│── entity
│── exception
│── dto 
│── config 

API Endpoints :
Employee APIs :
POST /api/employees → Create Employee,
GET /api/employees → Get All Employees,
GET /api/employees/{id} → Get Employee by ID,
PUT /api/employees/{id} → Update Employee,
DELETE /api/employees/{id} → Delete Employee
